# Сравнительный анализ сервисов геокодирования

## 🔬 Методология сравнения

### 1. Прямое геокодирование (Адрес → Координаты)

- **Эталон**: Яндекс.Геокодер (принят за базовый из-за 100% покрытия и высокой точности в РФ)
- **Метрика точности**: Расстояние по формуле гаверсинусов (метры)
- **Критерии оценки**:

| Расстояние до эталона | Оценка |
|----------------------|--------|
| < 50 м               | ✅ Точно |
| 50–200 м             | ⚠️ Допустимо |
| > 200 м              | ❌ Ошибка |

- **Дополнительные метрики**:
  - **Success Rate**: % запросов, вернувших координаты
  - **Median Error**: медиана погрешности (устойчива к выбросам)
  - **Avg Response Time**: среднее время ответа (мс)

### 2. Обратное геокодирование (Координаты → Адрес)

- **Эталон**: Исходный адрес из тестового набора
- **Метрика совпадения**: Семантическое сравнение после нормализации
- **Критерии оценки**:

| Тип совпадения | Пример | Оценка |
|---------------|--------|--------|
| 🟢 Полное | "Красная площадь, 1" = "Красная площадь, 1" | ✅ |
| 🟡 Частичное | "Красная площадь" ≈ "Красная пл, д. 9" | ⚠️ |
| 🔴 Не совпадает | "Красная площадь" ≠ "Тверская улица" | ❌ |

## 📋 Рекомендации по выбору сервиса

### ✅ Яндекс.Геокодер — выбирайте, если:
- Нужна максимальная точность для навигации и карт
- Работаете с визуализацией на карте (маркеры, маршруты)
- Критично точное позиционирование (< 50 м)
- Готовы мириться с более высокой стоимостью и лимитами

### ✅ DaData — выбирайте, если:
- Нужна массовая обработка адресов (очистка, нормализация, геокодирование)
- Важна скорость и экономия запросов
- Работаете с CRM, базами клиентов, формами ввода
- Нужна стандартизация по ФИАС/ГАР

### 🏆 Рекомендация для большинства бизнес-задач: **DaData**

> Точность ≈ Яндекс, скорость в 2–4× выше, дешевле, лучше для пакетной обработки.

### ❌ GigaChat — не рекомендуется для геокодирования, если:
- Нужны точные координаты или адреса
- Задача требует воспроизводимости и верифицируемости
- Работаете с продакшен-системами

### ⚠️ Можно использовать только для:
- Черновой разметки неструктурированных текстов
- Прототипирования и демонстраций
- Задач, где допустима погрешность > 500 м
